# Personal Portfolio Website

A responsive and modern portfolio website designed to showcase my projects, technical skills, certifications, achievements, and professional journey in Software Development, Artificial Intelligence, Machine Learning, and Data Analytics.

## Features

- Professional landing page
- About Me section
- Skills showcase
- Projects portfolio
- Certifications and achievements
- Contact information
- Responsive design for desktop and mobile devices

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Sections

### Home
Introduction and professional summary.

### About
Educational background and career objectives.

### Skills
Technical skills and tools.

### Projects
Featured projects including:
- DermaMind – AI Skin Cancer Detection System
- RetailShield – Retail Theft Detection System
- CureBot – AI Healthcare Chatbot

### Certifications
Professional certifications, internships, and virtual experience programs.

### Contact
Links to GitHub, LinkedIn, Email, and other professional platforms.

## Purpose

This portfolio serves as a central platform to showcase my academic projects, technical expertise, certifications, and achievements to recruiters, hiring managers, and collaborators.
